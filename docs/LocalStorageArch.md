# Data Structure in Local Storage

```
|- Posts
  |- Entities
    |- ... post detail
    |- LastUpdate <- Last update time for post
    |- Comment
      |- ...comment detail
      |- LastUpdate <- Last update time for comment
  |- LastUpdate  <- Last update time for post list
|- Tags
  |- Entities
    |- ...tag detail
  |- LastUpdate
|- User
  |- Entities
    |- ...user info
  |- LastUpdate
```
